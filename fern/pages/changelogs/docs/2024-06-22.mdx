# June 2024

## RSS and Atom Feeds
Added support for RSS and Atom feeds in documentation.

```yaml
# docs.yml
features:
  rss:
    enabled: true
    title: "API Updates"
    description: "Latest API changes and updates"
```

## JSON-LD Support
Added JSON-LD support for better SEO.

```html
<script type="application/ld+json">
{
  "@type": "TechArticle",
  "headline": "API Reference",
  "description": "Complete API documentation"
}
</script>
```

## Syntax Highlighting
Added support for BAML and Jinja syntax highlighting.

```jinja
{% for endpoint in api.endpoints %}
  {{ render_endpoint(endpoint) }}
{% endfor %}
```
