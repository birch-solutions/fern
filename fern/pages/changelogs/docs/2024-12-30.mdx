## Audio Streaming in API Explorer
Added support for streaming audio directly within the API Explorer. This feature enables testing audio endpoints without leaving the documentation. 

Check it out live in ElevenLabs' [API Explorer](https://elevenlabs.io/docs/api-reference/text-to-speech/convert?playground=/docs/api-reference/text-to-speech/convert-as-stream) to let users test text-to-speech endpoints and hear the results instantly.

## Table of Contents Control
Added granular control over table of contents visibility using frontmatter configuration. Documentation authors can now customize TOC display per page.

```yaml
---
title: API Reference
hide-toc: true
toc:
  minHeadings: 2
  maxDepth: 3
  exclude:
    - '#examples'
    - '#changelog'
---
```

## Feedback Collection
Enhanced the documentation feedback system with improved form handling and data collection. Teams can now gather more detailed feedback about their documentation.

```typescript
interface FeedbackSystem {
  categories: {
    helpful: boolean;
    clarity?: 'clear' | 'confusing' | 'incomplete';
    technical?: 'accurate' | 'outdated' | 'incorrect';
  };
  allowComments: boolean;
  notifyOn: ['negative', 'technical-issue'];
}
```

## Domain Path Resolution
Improved handling of documentation across multiple domains and path structures. Documentation now maintains proper navigation regardless of hosting configuration.

```typescript
const pathResolver = {
  mode: 'strict',
  rules: [
    {
      domain: 'docs.example.com',
      basePath: '/',
      versions: true
    },
    {
      domain: 'api.example.com',
      basePath: '/docs',
      versions: false
    }
  ],
  fallback: '/404'
}
```
