# October 2024

## JWT API Key Integration
Implemented automatic API key extraction from JWT tokens in the documentation playground. Users can now test authenticated endpoints more easily with automatic credential handling.

```typescript
const jwtConfig = {
  extractApiKey: true,
  tokenLocation: 'header',
  autoRefresh: true,
  schema: {
    apiKeyPath: 'payload.api_key',
    rolePath: 'payload.permissions'
  }
}
```

## Query Parameter Enhancement
Improved handling of query parameters in documentation middleware. Complex query parameters are now properly handled and displayed in the documentation.

```typescript
const queryConfig = {
  arrayFormat: 'brackets',
  allowDots: true,
  parseNumbers: true,
  parseBooleans: true
}
```

## Documentation Parsing System
Enhanced documentation parsing for special elements like asides and custom components. Documentation authors can now use richer formatting options.

```yaml
asides:
  types:
    - note
    - warning
    - tip
    - important
  styling:
    note:
      icon: 'info'
      color: 'blue'
    warning:
      icon: 'alert-triangle'
      color: 'amber'
```

## Host Query Handling
Improved handling of host information in query parameters for better middleware support. Documentation now handles complex hosting scenarios more reliably.

```typescript
const hostConfig = {
  appendHost: true,
  allowedHosts: [
    'api.example.com',
    '*.staging.example.com'
  ],
  rewriteRules: [
    { 
      pattern: 'localhost:*',
      rewrite: 'api.local'
    }
  ]
}
```
