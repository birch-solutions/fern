## C# Model Generation
Added support for generating C# models from your API definition.

```csharp
// Generated C# model
public class User
{
    [JsonPropertyName("id")]
    public string Id { get; set; }

    [JsonPropertyName("email")]
    public string Email { get; set; }
}
```

## License Configuration
Added ability to specify license information in SDK metadata.

```yaml
# generators.yml
sdk:
  license:
    type: MIT
    file: ./LICENSE
```
