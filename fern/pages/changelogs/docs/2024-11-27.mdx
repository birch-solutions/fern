## Auth token autocompletion in API Explorer

Auth tokens are now automatically injected into API explorer after authenticating to your API.

### Why it matters?
- Saves time for your users by automatically filling in API keys.
- Minimizes errors during authentication.

![API Explorer Splash Image](../../../../fern/images/changelogs/docs/api-explorer.png)

Check it out live in [Webflow's API Explorer](https://developers.webflow.com/data/reference/sites/list?playground=/data/reference/sites/list).
