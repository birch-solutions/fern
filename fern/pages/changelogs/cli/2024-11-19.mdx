## 0.45.0-rc47
**`(feat):`** OAuth Client Credential Auth Scheme now supports the ability to set token header 
and prefix fields for use with docs playground.
```yml api.yml
auth-schemes:
  OAuth:
    scheme: oauth
    type: client-credentials
    token-header: Fern-Authorization
    token-prefix: Fern-Bearer
    ...
```
