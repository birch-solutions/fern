## 1.0.0
**`(fix):`** The python SDK now supports specifying whether or not to follow redirects in requests by default, and exposes an option to override that functionality for consumers.

  ```yaml
  generators:
    - name: fernapi/fern-python-sdk
      config:
        follow_redirects_by_default: true
  ```

  which then just instantiates the client like so:

  ```python
  client = SeedExhaustive(
        token="YOUR_TOKEN",
        follow_redirects=True  # This is defaulted to the value passed into follow_redirects_by_default, and ignored if not specified
  )
  ```
