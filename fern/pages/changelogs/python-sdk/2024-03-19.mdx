## 0.12.4
**`(fix):`** Allow full forward compat with enums while keeping intellisense by unioning enum literals with `typing.AnyStr`.

  Before:

  ```python
  Operand = typing.Union[typing.AnyStr, typing.Literal[">", "=", "less_than"]]
  ```

  After:

  ```python
  Operand = typing.Literal[">", "=", "less_than"]
  ```
