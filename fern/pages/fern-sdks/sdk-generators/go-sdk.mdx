[![Go Generator Version](https://img.shields.io/docker/v/fernapi/fern-go-sdk)](https://github.com/fern-api/fern/blob/main/generators/go/sdk/VERSION) 

[Source code](https://github.com/fern-api/fern/tree/main/generators/go)<br />

The Go SDK generator outputs a fully functional Go SDK.

## Local

Dump the generated SDK to the local file system.

<CodeBlock title="generators.yml">
```yaml
groups:
  local:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.12.0
        output:
          location: local-file-system
          path: ../generated/sdk/go
```
</CodeBlock>

## Publish to GitHub 

Publish the generated SDK to GitHub.

<CodeBlock title="generators.yml">
```yaml
groups:
  publish:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.12.0
        github:
          repository: fern-petstore/petstore-go # replace package & org
```
</CodeBlock>
