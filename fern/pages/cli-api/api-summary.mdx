The Fern API allows you to manage SDKs and code snippets using Fern's public-facing RESTful API. API access is available on paid plans. [Reach out](https://buildwithfern.com/contact) to get started.

## Accessing the Fern API

Fern maintains official API clients for Node.js and Python. We recommend using these clients, though the API supports any language or framework that sends HTTP requests.

## Authentication

Fern API requests require a bearer token for authentication. Use the CLI command [`fern token`](/learn/cli-api/cli-reference/commands#fern-token) to generate a bearer token. Tokens do not expire.
