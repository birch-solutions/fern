---
title: fern format
description: The command "fern format" formats your Fern Definition.
---

Use `fern format` to format the YAML of your Fern Definition.

```bash
fern format
```

## Options

### api

Use `--api` to specify which API definition to format. 

```bash
fern format --api public-api
```

### ci

Use `--ci` to prompt failure with non-zero exit status if files are not formatted correctly. 

```bash
fern format --ci
```
