---
title: fern login
description: The command "fern login" will log you into GitHub with Fern. 
---

Use `fern login` to login to the Fern CLI via GitHub. Logging in allows you 
join GitHub organizations, gain permissions, and contribute to projects.

```bash
fern login
```

## Options

### device-code

Use `--device-code` to login via device code authorization. 

```bash
fern mock --device-code
```

<Note>
To enable CI/CD, use [`fern token`](/cli-api/cli-reference/commands/token).
</Note>
