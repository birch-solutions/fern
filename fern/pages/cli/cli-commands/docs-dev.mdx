---
title: fern docs dev
description: The command "fern docs dev" runs a local development server to preview docs.
---

Use `fern docs dev` to run a local development server to preview your docs.

```bash
fern docs dev
```

## Options

### port

Use `--port` to specify the port the docs preview will be run on. 

```bash
fern docs dev --port 57908
```
