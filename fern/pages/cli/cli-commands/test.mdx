---
title: fern test
description: The command "fern test" starts a mock server for an API. 
---

Use `fern test` to run SDK tests specified in `--command`. It runs a mock server in the background
that is terminated upon completion of the tests.

For more information on testing Fern SDKs, see [Integration Tests](/learn/sdks/features/integration-tests).

```bash
fern test --command <command>
```

## Options

### api

Use `--api` to specify the API to mock if you have multiple defined in your `fern/apis/` folder. 

```bash
fern test --api public-api
```
