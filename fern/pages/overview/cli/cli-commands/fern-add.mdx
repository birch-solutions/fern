---
title: Use fern add to use a generator
description: The command "fern add" includes a new generator in your generators configuration file.
---

Use `fern add <generator>` to include a new generator in your `generators.yml`.

See a [list of generators to choose from](/learn/sdks/overview).

## Usage

```bash
fern add <generator>
```

## Example

<CodeBlocks>
    <CodeBlock title="Node.js SDK">
    ```bash
    fern add fern-typescript-node-sdk
    ```
    </CodeBlock>

    <CodeBlock title="Python SDK">
    ```bash
    fern add fern-python-sdk
    ```
    </CodeBlock>

    <CodeBlock title="Go SDK">
    ```bash
    fern add fern-go-sdk
    ```
    </CodeBlock>

    <CodeBlock title="Postman Collection">
    ```bash
    fern add fern-postman
    ```
    </CodeBlock>

    <CodeBlock title="FastAPI">
    ```bash
    fern add fern-fastapi-server
    ```
    </CodeBlock>

</CodeBlocks>
