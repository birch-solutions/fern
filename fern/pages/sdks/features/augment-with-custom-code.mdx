The Fern generated SDKs are designed to be extended with custom code. Your custom code 
can be used to add additional functionality to the SDK and will live in harmony with
the generated code.

### Why augment with custom code?
- **Custom Helper Functions**: Add helper functions that are specific to your API
- **Chain Methods**: Chain methods to make the SDK more user-friendly

<Note>
  Anything you want to do with the SDK that is not generated by 
  Fern can be done with custom code.
</Note>

### How to augment with custom code

You can use inheritance (i.e. `extends`) to add extra functionality to the client. 
Fern's generated SDK classes all have protected variables that can be accessed by subclasses
for this reason. 

<Tabs>
  <Tab title="TypeScript">
    ### Adding a helper method to the SDK

    The easist way to add a helper method to the SDK is to export a function
    from a new file. Follow the steps below to export a helper function 
    (e.g. `myHelper()`) from the SDK.

    <Steps>
      ### Create a new file `src/helper.ts`

      ```typescript
      export function myHelper(movie: Movie): string {
        return `${movie.title} (${movie.year})`;
      }
      ```

      ### Add `src/helper.ts` to `.fernignore`

      <CodeBlock title=".fernignore">
        ```yaml
        #  Specify files that shouldn't be modified by Fern

        + src/helper.ts
        ```
      </CodeBlock>

      ### Consume the helper 

      ```typescript
      import { myHelper } from "sdk/helper";

      myHelper(movie);
      ```
    </Steps>

    ### Adding a custom method to the client

    If you would like to add a custom method to the client itself (e.g. `client.myHelper()`),
    then you can extend the client class and add the method there.

    <Note>
      See how FlatFile does this in their SDK [here](https://github.com/FlatFilers/flatfile-node/blob/main/src/wrapper/FlatfileClient.ts). 
    </Note>

    <Steps>
      ### Create a new file `src/wrapper/ImdbClient.ts`

      ```typescript
      import { ImdbClient as FernClient } from "../client";

      export class ImdbClient extends FernClient {
        
        public myHelper(movie: Movie): string {
          return `${movie.title} (${movie.year})`;
        }

      }
      ```

      ### Export the extended client

      Instead of exporting the generated client, export the extended client. To do this, 
      you will need to update the `index.ts` file. 

      <CodeBlock title="src/index.ts">
        ```typescript
        + export { ImdbClient } from src/wrapper/ImdbClient; // instead of `src/Client`
        ```
      </CodeBlock>

      ### Update `.fernignore`

      Add both the `wrapper` directory and `index.ts` to `.fernignore`.

      <CodeBlock title=".fernignore">
        ```yaml
        + src/wrapper
        + src/index.ts
        ```
      </CodeBlock>
    </Steps>
  </Tab>
</Tabs>
