---
title: Fern | Use Fern Definition to define your API
description: An overview of how to define your API using the Fern Definition format, then use Fern to generate SDKs and API reference documentation for your API.
---

Use Fern's own Fern Definition (Fern Def) format to define your API. Alternatively, you can use the [OpenAPI specification format](/overview/define-your-api/openapi/overview).

Fern generates SDKs and API reference documentation from your Fern Def. Create a `fern/` project through one of our [Quickstarts](/overview/get-started) then update the Fern Def files in `fern/definitions/` with your own. 

Configure your API with the [api.yml file](/overview/define-your-api/ferndef/api-yml-reference).

Run [fern generate](/overview/cli/cli-commands/fern-generate#for-sdks) to generate SDKs or [fern generate --docs](/overview/cli/cli-commands/fern-generate#for-documentation) to generate docs. 

Need an OpenAPI spec to work with OpenAPI tools? Use our [OpenAPI generator](/fern-sd-ks/other-generators/spec/openapi/) from your Fern Def.
