[Source code](https://github.com/fern-api/fern-python)

**Latest version**: `0.7.2`

The Python SDK generator outputs a fully functional Python
SDK. It can publish the SDK to [pypi.org](pypi.org) (or any other PyPI repository).

## Local

Dump the generated SDK to the local file system. Available on the open source plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    local:
      - name: fernapi/fern-python-sdk
        version: 0.7.2
        output:
          location: local-file-system
          path: ../../generated/sdk/python
```
</CodeBlock>

## Publish an Internal Package

Publish the generated SDK to a private PyPI repository hosted by Fern. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    internal:
      - name: fernapi/fern-python-sdk
        version: 0.7.2
        output:
          location: pypi.buildwithfern.com
          package-name: imdb # replace with your package name
```
</CodeBlock>

## Publish a Public Package

Publish the generated SDK to PyPI. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    public:
      - name: fernapi/fern-python-sdk
        version: 0.7.2
        output:
          location: pypi
          package-name: imdb
          token: ${PYPI_TOKEN}
        github:
          repository: imdb/imdb-python    
```
</CodeBlock>
