title: The Rust SDK generator
description: Use Fern's Rust SDK generator and your API definition to output a fully functional Rust SDK (client libraries) to your local filesystem, or publicly to crates.io with Cargo.
---

<Callout intent="tip">
Coming in 2024!
</Callout>

<br />

Output a fully functional Rust SDK. Publish the SDK to [crates.io](https://crates.io) with [Cargo](https://doc.rust-lang.org/cargo/) or to a private repository hosted by Fern.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-rust-sdk
  version: 0.0.0
```
</CodeBlock>
