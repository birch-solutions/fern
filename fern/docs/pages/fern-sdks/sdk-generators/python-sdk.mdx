---
title: The Python SDK generator
description: Use Fern's Python SDK generator and your API definition to output a fully functional Python SDK to your local file system, to a private repository, or to PyPI.
---

[Source code](https://github.com/fern-api/fern/tree/main/generators/python)<br />
**Latest version**: `0.8.0`

Output a fully functional Python SDK. The generator can publish the SDK to [the Python Package Index at PyPI.org](https://pypi.org), or to a private repository hosted by Fern. You can also download the SDK to your local file system.

## Publish a public package

Publish the generated SDK to PyPI. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  public:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.x.x # declare your desired version
        output:
          location: pypi
          package-name: petstore # replace with your package name
          token: ${PYPI_TOKEN}
        github:
          repository: petstore/petstore-python # replace package & org
```
</CodeBlock>

## Publish an internal package

Publish the generated SDK to a private PyPI repository hosted by Fern. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  internal:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.x.x # declare your desired version
        output:
          location: pypi.buildwithfern.com
          package-name: petstore # replace with your package name
```
</CodeBlock>

## Local

Download the generated SDK to the local file system.

<CodeBlock title="generators.yml">
```yaml
groups:
  local:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.x.x # declare your desired version
        output:
          location: local-file-system
          path: ../sdks/python
```
</CodeBlock>
