---
title: The PHP SDK generator
description: Use Fern's PHP SDK generator and your API definition to output a fully functional PHP SDK to your local file system, to a private repository, or to Packagist.
---

<Callout intent="tip">
Coming in 2024!
</Callout>

<br />

Output a fully functional PHP SDK. The generator can publish the SDK to [Packagist.org](https://Packagist.org), or to a private repository hosted by Fern. You can also download the SDK to your local file system.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-php-sdk
  version: 0.0.0
```
</CodeBlock>
