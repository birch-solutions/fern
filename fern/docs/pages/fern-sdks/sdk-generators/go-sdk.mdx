---
title: The Go SDK generator
description: Use Fern's Go SDK generator and your API definition to output a fully functional Go SDK to your local file system, or publish it to GitHub. 
---

[Source code](https://github.com/fern-api/fern/tree/main/generators/go)<br />
**Latest version**: `0.12.0`

Output a fully functional Go SDK. The generator can publish the SDK to [GitHub](https://www.github.com). You can also download the SDK to your local file system.

## Publish to GitHub 

Publish the generated SDK to GitHub.

<CodeBlock title="generators.yml">
```yaml
groups:
  publish:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.x.x # declare your desired version
        github:
          repository: fern-petstore/petstore-go # replace package & org
```
</CodeBlock>

## Local

Download the generated SDK to your local file system.

<CodeBlock title="generators.yml">
```yaml
groups:
  local:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.x.x # declare your desired version
        output:
          location: local-file-system
          path: ../sdks/go
```
</CodeBlock>
