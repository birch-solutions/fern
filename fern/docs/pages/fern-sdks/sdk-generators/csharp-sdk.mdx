---
title: The C# SDK generator
description: Use Fern's C# SDK generator and your API definition to output a fully functional C# SDK to your local file system, to a private repository, or to NuGet.
---

<Tip>
Generally available in 2024! If you'd like to join the private beta, [drop us an email](mailto:support@buildwithfern.com?subject=Join%20the%20C%23%20private%20beta).
</Tip>

[Source code](https://github.com/fern-api/fern/tree/main/generators/csharp)<br />
**Latest version**: `0.0.0-1`

<br />

Output a fully functional C# SDK. The generator can publish the SDK to [NuGet.org](https://www.nuget.org), or to a private repository hosted by Fern. You can also download the SDK to your local file system.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-csharp-sdk
  version: 0.0.0-1
```
</CodeBlock>
