Add a code generator to `generators.yml`.

## Usage

```bash
fern add <generator-name>
```

## Examples

Add the TypeScript SDK generator:

```bash
fern add typescript
```

Add the Java Spring server generator:

```bash
fern add java-spring-server
```

Add the Postman collection generator:

```bash
fern add postman
```
